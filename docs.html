API Response Status

Code Description
200 OK
400 ข้อมูลคำขอที่ส่งมาไม่ถูกต้อง
401 ไม่มีสิทธิในการใช้งาน
404 ไม่พบข้อมูล
500 ระบบมีปัญหา
▶ True API: 
GET api/search?token={token}&type=phone&value={value}
  Content-Type: application/json
  {
    "param": "value"
  }
ตัวอย่าง Response:
{
    "status": "success",
    "type": "phone",
    "value": "0909466495",
    ...
  }
ตัวอย่าง Error:
{
    "error": "Unauthorized: Invalid or missing API token"
  }
